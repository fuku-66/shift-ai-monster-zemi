🥚 プロジェクトの目的

みんなで1体のモンスターを育てよう。
あなたの提出が、モンスターの「教科の傾向」を形作ります。
数学が多ければゴーレム、英語が多ければフェニックス…バランスよく育てれば伝説のユニコーンに進化！

🚶 提出の流れ

  1. 「課題一覧」から、挑戦する課題を選ぶ
  2. AIを使って課題に取り組む
  3. 「提出する」ボタン → Googleフォームに記入して送信
  4. 講師がチェックして「承認」 → モンスターが育つ！Discordに通知が届く

🐉 進化の仕組み（4段階）

Lv姿累計XP備考
1🥚 タマゴ0〜199 XP全員共通
2🐣 ヒナ200〜599 XP全員共通
3子モンスター600〜1499 XP最強の軸の子モンスターになる
4最終形態1500 XP〜5軸が均等なら伝説のユニコーン神獣に！

軸ごとの最終形態

  • 🤖 AI基礎が最強 → サイバービースト
  • 📖 英語が最強 → フェニックス
  • 📐 数学が最強 → クリスタルゴーレム
  • 📚 勉強法が最強 → シャドウウルフ
  • 🎨 クリエイティブが最強 → レインボードラゴン
  • 🦄 5軸の差が10以下（バランス） → ユニコーン神獣（レア）

⭐ XP加算ルール

  • ★1（やさしい） … +15 XP
  • ★2 … +25 XP
  • ★3（ふつう） … +40 XP
  • ★4 … +60 XP
  • ★5（チャレンジ） … +80 XP

XPは 選んだ教科の軸だけ に加算されます。
例：英語の★3課題を出す → 英語軸に +40 XP

📝 提出時のヒント

  • スクショ1枚＋コメント数行でOK。完璧を目指さなくていい
  • AIにやらせるだけじゃなく、自分の言葉で「気づいたこと」を一言添えるとGood
  • 同じ課題に何度挑戦してもOK（ただし内容は変えよう）
  • 差し戻しは「もっと良くなるヒント」がもらえるチャンス
