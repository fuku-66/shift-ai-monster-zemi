## 1. プロジェクト概要

### 1.1 目的
SHIFT AI ジュニア（中高生）の学習アウトプットを可視化し、ゲーミフィケーションで継続動機を生み出す。

### 1.2 参考サイト
https://ryonihonyanagi-cloud.github.io/agent-zemi/

### 1.3 コンセプト
「**みんなで1体のドラゴンを育てる**」
- 学習アウトプットを提出 → 講師承認 → XP加算 → ドラゴンがLv.UP
- ログイン不要、全員共通のドラゴンを育成
- 異世界転生チート系の世界観

## 4. 機能要件

### 4.1 ドラゴン育成システム

#### 進化段階（5段階）
| Lv | 名称 | 画像ファイル | 必要XP（累積） |
|----|------|------------|--------------|
| 1 | タマゴ | `lv1-egg.png` | 0 |
| 2 | ベビードラゴン | `lv2-baby.png` | 100 |
| 3 | 若竜 | `lv3-young.png`（未作成） | 400 |
| 4 | 成竜 | `lv4-adult.png`（未作成） | 1100 |
| 5 | 神竜（Max） | `lv5-god.png`（未作成） | 2400 |

#### XP算出ルール
- 1提出（承認済み）= **+20 XP**
- 難易度ボーナス・教科ボーナスは Phase 1 では適用しない

#### 試算
- 15名 × 週3回提出 × 20XP = 900 XP/週
- 2400 XP ÷ 900 = **約2.7週でLv.Max**（目標1ヶ月）

### 4.2 提出フロー
```
1. 生徒: サイト「提出する」ボタン → Googleフォーム
2. Googleフォーム送信 → スプシに自動記録（未承認ステータス）
3. mariko: スプシで✅承認 or ❌差し戻し（フィードバック記入）
4. GAS トリガー:
   - 承認 → XP +20 加算 + Discord通知
   - 差し戻し → Discord通知（フィードバック付き）
5. サイト: GAS WebApp 経由でスプシを読み、ドラゴン状態を表示
```

### 4.3 サイト構成（3ページ）

| ページ | URL | 内容 | 重さ対策 |
|--------|-----|------|---------|
| トップ | `/` | ドラゴン表示・XPバー・最新5件・提出ボタン | 最新5件のみ |
| アーカイブ | `/archive.html` | 全提出一覧・教科フィルタ・ページ分割 | 20件/ページ |
| 使い方 | `/how-to.html` | ルール・XP計算式・Lv進化表 | 静的HTML |

## 5. 非機能要件

### 5.1 パフォーマンス
- トップページ表示: 2秒以内
- アーカイブ1ページ: 20件表示でサクサク
- 提出件数が1000件超えても重くならない設計

### 5.2 可用性
- GitHub Pages の稼働率に依存（実質24/365）
- GAS WebApp の呼び出し上限に注意

### 5.3 セキュリティ
- 個人情報は収集しない（Discord名・ニックネームのみ）
- ログインなし = 認証情報不要
- 講師承認で荒らし・不正XP加算を防止

## 6. 技術スタック

| レイヤー | 技術 | 選定理由 |
|---------|------|---------|
| ホスティング | **GitHub Pages** | 0円・参考サイトと同構成・mariko個人リポジトリ |
| フロント | HTML + CSS + Vanilla JS | フレームワーク不要・軽量 |
| バックエンド | **GAS Web App**（Apps Script） | スプシ連携が最速・既存資産活用 |
| データ | **Googleスプレッドシート** | 承認オペレーションが簡単 |
| 提出フォーム | **Googleフォーム** | 中高生に馴染みある・スプシ連携 |
| 通知 | **Discord Webhook** | SHIFT AIジュニアで使用中 |
| 画像 | GPT Image 2（ドット絵生成） | marikoが手動生成、スタイル統一可 |
| アニメ | CSS `@keyframes` | 軽量・差し替え容易 |

## 7. ビジュアル・デザイン

### 7.1 世界観
- 異世界転生チート系
- ドット絵によるレトロRPG風
- 紫×ゴールドのカラーパレット（ドラゴンの配色と統一）

### 7.2 カラーパレット（暫定）
- Primary: 紫系（`#7D3C98` 等）
- Accent: ゴールド系（`#F1C40F` 等）
- Background: 黒 or ダークネイビー（レトロRPG感）
- Text: 白 or 淡い色

### 7.3 アニメーション方針
- Phase 1: CSS `@keyframes` で擬似アニメ（呼吸・揺れ・進化エフェクト）
- Phase 2: 2〜4フレームGIFに差し替え（画像パス変数化で対応済み）

## 8. 運用ルール

### 8.1 講師承認の運用
- mariko が1日1回スプシをチェックして承認 or 差し戻し
- 差し戻し時は「どう改善すれば良いか」のフィードバックを必ず記入

### 8.2 Discord 通知先
- 運用中の1チャンネルに集約
- 承認通知：「🐲 Lv.UP！ 現在Lv.◯ / ◯◯ XP」
- 差し戻し通知：「📝 〇〇さんの提出にフィードバック: ○○」
