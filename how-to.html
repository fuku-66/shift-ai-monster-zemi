🥚 プロジェクトの目的

みんなで1体のモンスターを育てよう。
あなたの提出が、モンスターの「教科の傾向」を形作ります。
どの教科を頑張るかで、進化する姿が変わってきます。

✨ どんな姿に進化するかは、提出してからのお楽しみ。
隠しモンスターも存在するらしい…

📋 2つのカテゴリ

📦 成果物（Output）

自分で作ったもの。AIで生成・制作したもの。

例: AI画像、AI文章、スライド、動画、デザイン、コード

🏆 成果（Result） — XP×3倍

結果として得た実績。

例: コンテスト入賞、受賞、外部評価、採用、成績アップ、点数アップ

💡 迷ったら：「自分で作った」→📦 成果物 / 「外から評価された」→🏆 成果

🚶 提出の流れ

📦 成果物の場合

  1. 「課題一覧」から、挑戦する課題を選ぶ
  2. AIを使って課題に取り組む
  3. 「📝 提出する」ボタン → Googleフォームに記入して送信
  4. 送信時に即XP加算！モンスターが育ち、Discordに通知

🏆 成果の場合

  1. 課題一覧ページの「🏆 成果を報告する」ボタンを押す（or ホームから）
  2. タイトル・教科・難易度・経緯を記入
  3. 受賞ページや成績表のURL・写真を添付（任意・あると説得力UP）
  4. 送信時に即XP×3倍加算！Discordで成果報告がアナウンスされる

🐉 進化の仕組み（4段階）

Lv姿累計XP備考

⭐ XP加算ルール

📦 成果物（難易度で変動）

難易度XP
★1 やさしい+15 XP
★2+25 XP
★3 ふつう+40 XP
★4+60 XP
★5 チャレンジ+80 XP

🏆 成果（一律）

+120 XP（難易度問わず一律）

XPは 選んだ教科の軸だけ に加算されます。
例：英語の★3 成果物を提出 → 英語軸に +40 XP
例：英語の成果（英検3級合格など）を報告 → 英語軸に +120 XP

📝 提出時のヒント

  • スクショ1枚＋コメント数行でOK。完璧を目指さなくていい
  • AIにやらせるだけじゃなく、自分の言葉で「気づいたこと」を一言添えるとGood
  • 同じ課題に何度挑戦してもOK（ただし内容は変えよう）
  • 🏆 成果報告は参考URL・スクショがあると伝わりやすい
  • ⚠️ 提出内容はDiscord・サイト内アーカイブに公開されます。フルネーム・住所・学校名・電話番号などの個人情報は記入しないでください（運営側で確認・修正します）
  • 不適切な提出（コピペ・空っぽ）は後から取り消されてXPが減ることがあるので注意
